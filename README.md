# AutoSplit-Penumbra
Auto Split Live Split exe


-Auto Split
-Auto Pause
-Auto Reset
-Player position 
-Next Trigger or Door Distance

Ce programme utilise les adresses de penumbra pour communiquer avec live Split en utilisant Les touches qui sont dans Setting.Cfs de LiveSp.
Pour que le programme puisse utiliser plus facilement live split je vous conseille d'utiliser des lettres comme touche de reset pause et split.
Au lancement du programme si les touches que vous avez modifié ne corresponde pas c'est que vous n'avez pas redémarre live Split.
En jeu si la position du joueur n'est pas détecter c'est que le programme n'est pas en administrateur il ne peut donc pas lire les adresses de penumbra.
